Projects

I'm developing two web apps using R and a web game using Java.

1) One of the web apps is used to take in an excel file where the user is then given optins on how they want to filter their data. The user is allowed to choose which column they would like their data to be filtered by. The user is given three text boxes where one can type in strings of what they would like their data to be filtered by. Then a download button can be pressed to export an excel workbook containing a sheet with their original data, and up to three other sheets with their filtered data.

2) The other web app takes in an excel file and allows the user to choose which columns they would like to plot as their x and y variables. There are options for five different plot types, color types, and a text box to type in a title for the plot.

3) My web game is based on the tutorial videos from RealTutsGML on youtube. It is a platformer game in its infancy and I plan to create about 15 - 20 levels of varying difficulty for the game.

When I'm done with my projects I will either include them on my website or give links to them. I plan to realease initial fully functioning versions and update them with added features.
