Projects

Measure Data Cleaner

While interning at Autoliv, I developed a web application using the "shiny" package in R. The purpose of the application is to clean data that is received from a specific test that is often performed on the sensors we work with. I first wrote a script to automate this, but I gave the program a user interface to make it easier for others to use. The names of the csv files that are used as input for the program are denoted by "MeasData_####", hence why I called the application Measure Data Cleaner.

Auto Loan Calculator

I designed an auto loan calculator using the "shiny" package for R once again. I created this for a friend who sells cars, and it served as useful practice for me as well.
